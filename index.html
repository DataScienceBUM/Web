Proyecto Insignia IusTrack · En desarrollo

Sistema Judicial — IusTrack

Nació como un bot monolítico en R (v1.0, actualmente en producción) y está evolucionando hacia IusTrack: un producto privado de software construido como una API REST multi-instancia con Node.js y Playwright. El rediseño transforma un script de automatización en un sistema que puede servir a múltiples usuarios en simultáneo con endpoints documentados y arquitectura orientada a servicios.
